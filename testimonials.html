This file controls the styling for all pages. - Place it in the root folder or inside a `/css/` folder if you prefer. #### ✅ **JavaScript File (`script.js`, Optional)** - If you want to add JavaScript for animations or interactivity, place `script.js` in the root folder or inside a `/js/` folder. #### ✅ **Images Folder (`images/`)** - All images should be stored in a dedicated `/images/` folder. - Use **descriptive names** for easy reference (e.g., `doctor1.jpg`, `blog1.jpg`, `hero-image-1.jpg`). --- ### **How to Set It Up?** 1️⃣ **Create a folder named** `Prince_Princess_Clinic`. 2️⃣ **Inside the folder, create** `index.html`, `about.html`, `services.html`, `team.html`, etc. 3️⃣ **Create an `images/` folder and store all image files inside it.** 4️⃣ **Create `style.css` inside the root folder.** 5️⃣ **If needed, create `script.js` in the root folder or `/js/` folder.** --- ### **How to Open the Website?** ✅ **Simply double-click `index.html`** to open it in a browser. ✅ If you're using a **local server (XAMPP, WAMP, or VS Code Live Server)**, place the project folder inside `htdocs` and start the server. --- That’s everything! 🎉 Let me know if you need any modifications or help with anything else. 🚀
